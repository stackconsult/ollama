title: CLI Reference
---

### Run a model

```
ollama run gemma3
```

#### Multiline input

For multiline input, you can wrap text with `"""`:

```
>>> """Hello,
... world!
... """
I'm a basic program that prints the famous "Hello, world!" message to the console.
```

#### Multimodal models

```
ollama run gemma3 "What's in this image? /Users/jmorgan/Desktop/smile.png"
```

### Generate embeddings

```
ollama run embeddinggemma "Hello world"
```

Output is a JSON array:

```
echo "Hello world" | ollama run nomic-embed-text
```

### Download a model

```
ollama pull gemma3
```

### Remove a model

```
ollama rm gemma3
```

### Copy a model

```
ollama cp gemma3 my-gemma3
```

### Tag a model

Create a new tag/reference for an existing model:

```
ollama tag gemma3 my-gemma3-backup
```

This is useful for:
- Creating backups before making changes
- Versioning models with meaningful names
- Creating aliases for easier access

> Note: Tagging creates a new reference to the same underlying model data, so it's storage-efficient.

### List models

```
ollama ls
```

### Sign in to Ollama

```
ollama signin
```

### Sign out of Ollama

```
ollama signout
```

### Create a customized model

First, create a `Modelfile`

```
FROM gemma3
SYSTEM """You are a happy cat."""
```

Then run `ollama create`:

```
ollama create -f Modelfile
```

### List running models

```
ollama ps
```

### Stop a running model

```
ollama stop gemma3
```

### Start Ollama

```
ollama serve
```

To view a list of environment variables that can be set run `ollama serve --help`
